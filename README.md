2. Identify top-performing categories.
3. Generate business insights.
4. Provide recommendations based on the analysis.

## Methodology
- Data collection
- Data review and cleaning
- Exploratory Data Analysis (EDA)

## Key Findings
- Technology category shows the strongest performance.
- Furniture performs well but has room for improvement.
- Office Supplies contributes comparatively less.

The project demonstrates how data visualization helps organizations make informed business decisions.
